# Bot Patch Notes v0.3.6

## 1. Changes / Improvements

- Bot melee reach now matches the equipped weapon (spears/polearms reach farther, daggers shorter) instead of a fixed distance

## 2. Bug Fixes

- Fixed bots using an AoE skill (e.g. Slash Blast) on a single monster instead of the stronger single-target skill (e.g. Power Strike)
